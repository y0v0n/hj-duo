# 혜정 愛 듀오 - 프로필 관리 가이드

## 📁 파일 구성

이제 프로필 데이터가 3개의 파일로 분리되었습니다:

1. **main.html** - 메인 웹페이지 (스타일과 기능)
2. **female-profiles.js** - 여성 후보자 프로필 데이터
3. **male-profiles.js** - 남성 후보자 프로필 데이터

## 🚀 사용 방법

### 1. 파일 배치
세 파일을 **같은 폴더**에 넣어주세요:
```
📂 your-folder/
  ├── main.html
  ├── female-profiles.js
  └── male-profiles.js
```

### 2. 웹페이지 열기
`main.html` 파일을 더블클릭하거나 웹브라우저로 열면 됩니다.

## ✏️ 프로필 수정/추가 방법

### 여성 프로필 추가하기
`female-profiles.js` 파일을 텍스트 에디터로 열고, 배열 끝에 새 프로필을 추가하세요:

```javascript
const femaleProfiles = [
    // 기존 프로필들...
    {
        id: 'f8',
        name: '여자 8호',
        age: '94 년생',
        job: '디자이너',
        location: '서울 성동구',
        hobby: '드로잉, 카페투어',
        height: '163cm',
        religion: '무교',
        smoking: '비흡연',
        personality: '조용하지만 따뜻한 성격입니다.',
        intro: '',
        idealAge: '92-97',
        idealSmoking: '비흡연',
        idealType: '배려심 있고 책임감 있는 분',
        contact: '주선자: 홍길동'
    }
];
```

### 남성 프로필 추가하기
`male-profiles.js` 파일도 동일한 방식으로 수정하면 됩니다.

## 📝 프로필 필드 설명

각 프로필 객체는 다음 필드들을 포함합니다:

- **id**: 고유 식별자 (예: 'f1', 'm1')
- **name**: 이름 또는 별칭
- **age**: 나이 또는 출생년도
- **job**: 직업
- **location**: 거주지
- **hobby**: 취미
- **height**: 키
- **religion**: 종교
- **smoking**: 흡연 여부
- **personality**: 성격 소개 (HTML 태그 `<br>` 사용 가능)
- **intro**: 추가 소개 (선택사항)
- **idealAge**: 이상형 나이
- **idealSmoking**: 이상형 흡연 조건
- **idealType**: 이상형 설명
- **contact**: 연락처 정보

## ⚠️ 주의사항

1. **따옴표 주의**: 텍스트에 작은따옴표(')가 있으면 에러가 날 수 있습니다.
2. **쉼표 확인**: 각 프로필 객체 사이에는 쉼표(,)가 필요합니다.
3. **마지막 항목**: 배열의 마지막 프로필 뒤에는 쉼표를 붙이지 않습니다.
4. **파일 위치**: 세 파일이 반드시 같은 폴더에 있어야 합니다.

## 💡 팁

- 프로필을 삭제하려면 해당 프로필 객체 전체를 지우면 됩니다.
- 프로필이 없으면 "아직 등록된 후보가 없습니다" 메시지가 표시됩니다.
- 변경사항은 브라우저를 새로고침(F5)하면 바로 반영됩니다.
